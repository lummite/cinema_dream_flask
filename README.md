# Cinema Dream 🎬

Курсовая работа по дисциплине «Веб-проектирование».  

---

## О проекте

**Cinema Dream** — веб-приложение «Онлайн кинотеатр» для поиска и просмотра фильмов, сериалов и аниме. Пользователь может листать каталог по категориям, искать контент по названию, фильтровать и сортировать результаты, а также смотреть через встроенный плеер с метаданными.

---

## Стек технологий

| Слой | Технология |
|------|-----------|
| Разметка | HTML5 |
| Стили | SCSS → CSS |
| Клиентская логика | JavaScript |
| Сервер | Python, Flask |
| Шаблонизатор | Jinja2 |
| База данных | Microsoft SQL Server (SSMS) |

## Быстрый старт

```bash
# Создать и активировать виртуальное окружение
python -m venv venv
venv\Scripts\activate      # Windows
source venv/bin/activate   # Linux/macOS

# Установить зависимости
pip install flask pyodbc

# Скомпилировать SCSS (если нужно)
sass static/scss/style.scss static/css/style.css

# Запустить сервер
python app.py
```

## База данных

7 таблиц в MS SQL Server:

- **Genres** — справочник жанров
- **Movies** — фильмы (метаданные + путь к файлу)
- **MovieGenres** — связь «многие ко многим» фильмы ↔ жанры
- **Series** — сериалы и аниме (поле `type` = «Сериал» / «Аниме»)
- **SeriesGenres** — связь «многие ко многим» сериалы ↔ жанры
- **Seasons** — сезоны сериала (сезон, год)
- **Episodes** — эпизоды сезона (номер, название, длительность, путь к файлу)

## Функциональность

- **Поиск** — по названию через JS + Flask API
- **Фильтрация** — по году, жанру, рейтингу
- **Сортировка** — по названию и году (по возрастанию / убыванию)
- **Адаптивность** — 5 брейкпоинтов: ПК (≥1367px), планшет portrait/landscape, смартфон portrait/landscape
- **Кроссплатформенность** — протестировано в Yandex Browser, Firefox, Google Chrome
